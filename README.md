# Nutry.X

Site vitrine et interface client pour une offre de nutrition sportive premium.

L'application présente les plats signature, un panier, un espace profil athlète, une page contact, des informations légales et une interface admin.

## Lancer le projet

```bash
npm install
npm run dev
```

Le site est disponible sur `http://localhost:3000/`.

## Scripts

- `npm run dev` : lance Vite en local
- `npm run lint` : vérifie TypeScript
- `npm run build` : génère la version production
