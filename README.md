# Nutry.X

Interface web responsive dédiée à la nutrition sportive premium. Le projet réunit la présentation des plats, le parcours de commande et les espaces privés athlète et administrateur.

## Fonctionnalités

- Catalogue de plats avec macros et panier
- Profil athlète et suivi biométrique
- Messagerie directe athlète-administration
- Formulaire de contact et conciergerie
- Console admin avec gestion des plats, athlètes et commandes
- Import et optimisation locale des photos de plats
- Thèmes sombre et clair pour l’administration

## Technologies

React 19, TypeScript, Vite, Tailwind CSS, GSAP, Motion et Lucide.

## Installation

```bash
npm install
npm run dev
```

L’application est disponible sur [http://localhost:3000](http://localhost:3000).

## Commandes

```bash
npm run lint
npm run build
npm run preview
```

## Important

Cette version est une démonstration front-end. Les comptes, messages et commandes sont enregistrés dans le `localStorage` du navigateur. Une mise en production nécessite un backend, une base de données et une authentification serveur.
